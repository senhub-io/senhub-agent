### Microsoft Word Version

A consolidated Word document is available for offline reading and distribution.

**Generate the Word document:**
```bash
cd docs/user-guide
./generate-word-doc.sh
```

**Output:**
- SenHub-Agent-User-Guide-Complete.docx (~1.5 MB)
- Includes all 8 documentation chapters
- Automatic table of contents with numbered sections
- **32 Mermaid diagrams converted to PNG images** (architecture, flows, decision trees)
- Internal cross-reference links (all links work within the document)
- Professional document structure with title page
- Metadata (title, subtitle, author, date)

**Features:**
- Numbered sections (1. Installation, 2. Operating Modes, etc.)
- Three-level table of contents automatically generated by Pandoc
- All cross-file Markdown links transformed to internal anchors
- All Mermaid diagrams rendered as high-quality PNG images
- Clean professional format suitable for distribution and printing

**Requirements:**
- Pandoc: `brew install pandoc` (macOS)
- Mermaid CLI: `npm install -g @mermaid-js/mermaid-cli`

**Processing pipeline:**
1. Remove local "Table of Contents" sections from each file
2. Transform cross-file links to internal anchors
3. Extract Mermaid diagram code from markdown blocks
4. Convert each diagram to PNG using mermaid-cli
5. Replace Mermaid blocks with image references
6. Generate Word document with single global TOC and embedded images

**Document structure:**
- Single global table of contents (generated by Pandoc)
- No intermediate/local TOC sections
- Clean chapter flow from Installation through Troubleshooting
- All diagrams rendered as images
- All cross-references working as internal links
