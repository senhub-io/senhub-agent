# Archive - Notes de développement

Ce répertoire contient des notes de développement, analyses et historiques de fixes qui ont été archivés.

## Structure

- `citrix-dev-notes/` - Notes de développement et analyses du probe Citrix

## Pourquoi archiver ?

Ces documents ont été déplacés ici pour :
- **Clarté** : Séparer la documentation utilisateur des notes de développement
- **Historique** : Conserver l'historique des décisions et analyses techniques
- **Organisation** : Garder la documentation principale propre et à jour

## Note

Ces fichiers sont conservés à titre d'archive historique. Pour la documentation à jour, consultez :
- Documentation utilisateur : `/docs/user-guide/`
- Documentation technique : `/docs/technical-reference/`
- Documentation probes : `/docs/probes/`
